# 🌍 AI Disaster Text Detection

An AI-powered Natural Disaster Text Detection system built using Machine Learning and Natural Language Processing (NLP).  
The model analyzes text such as tweets, news headlines, or messages and predicts whether they are disaster-related.

---

## 🚀 Features

- 🔍 Detect disaster-related text
- 📊 Confidence score prediction
- 🔥 Disaster probability meter
- ⚡ Interactive web interface
- 🧠 TF-IDF text vectorization
- 🤖 Logistic Regression classifier

---

## 🛠 Tech Stack

- Python
- Scikit-learn
- Pandas
- Flask
- HTML / CSS / JavaScript
